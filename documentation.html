📌 Project Overview

CoverCraft is a pure front-end web application that generates professional PDF cover pages directly in the browser. Built with HTML5, CSS3, and JavaScript, it uses the jsPDF library to create print-ready A4 documents with precise measurements following MS Word specifications.

The application requires no server-side processing - all PDF generation happens locally in the user's browser, ensuring privacy and fast performance.

🎯 Key Features

  • Two Generator Versions: Standard cover page and Final cover page with dual logos
  • Live Preview: Real-time PDF preview inside an iframe
  • One-Click Download: Export as high-quality A4 PDF
  • Logo Upload: Support for custom college logos (resized automatically)
  • Optional Fields: Date field appears only when filled
  • Responsive Design: Works on desktop, tablet, and mobile devices
  • MS Word Margins: 1" top/bottom, 1.25" left/right
  • Times New Roman Font: Professional academic appearance

🛠️ How to Use

  1. Choose either Standard Cover or Final Cover from the navigation menu
  2. Fill in all required fields (College Name, Title, Student details, Teacher details)
  3. Upload logos (PNG/JPG format recommended)
  4. For Final Cover, optionally upload a decorative project line image
  5. Click Preview PDF to see the generated cover page inside the iframe
  6. Review the layout, formatting, and spacing
  7. Click Download PDF to save the cover page to your device
  8. The date field is optional - leave empty to omit from PDF

⚙️ Customization Guide

To modify the generator settings, edit the constants in the JavaScript section of the HTML file:

// Page margins (in mm)
const MARGIN_MM = 25.4;        // Change for different margins

// Font settings
const COLLEGE_FONT_SIZE_PT = 18;   // Increase for larger college name
const TITLE_FONT_SIZE_PT = 18;     // Change title font size

// Logo dimensions (Final Cover)
const LOGO_SIZE_MM = 30;           // Change logo size

// Line spacing
const BODY_LEADING_PT = 21;        // Change to 14 for single spacing

// Column gap
const COLUMN_SPACING_MM = 12 * 0.352778;  // Increase for wider gap

// Project line image width
const maxWidth = 80;               // Change in buildCoverPDF function

Key file locations:

🖼️ Default Images Setup

The Final Cover Page generator supports default images that load automatically when no file is uploaded:

  • project-line.png - Place in the same folder as project2.html for a default decorative line image
  • Supported formats: PNG, JPEG
  • If no default image is found, a dashed fallback line is drawn

⚠️ Troubleshooting

  • Logo not appearing? - Use the upload button instead of local file paths. Uploaded images work 100% of the time.
  • CORS errors? - Always use the file upload feature rather than trying to load local files directly.
  • Preview not showing? - Click "Preview PDF" after filling all fields. If still not working, check browser console for errors.
  • Text overlapping? - Adjust the spacing values in the constants section (SUB_LEADING_MM, TITLE_LEADING_MM, etc.)
  • Logo size issues? - Modify LOGO_WIDTH_MM and LOGO_HEIGHT_MM constants.
