📖 How CoverCraft Works

CoverCraft is a pure front‑end tool that generates custom PDF cover pages using jsPDF. No server, no uploads – everything runs locally in your browser.

🚀 Launch Cover Page Generators:
📄 Standard Cover Page 🏛️ Final Cover Page (Two Logos + Project Line)

✨ Features

  • Custom title, subtitle, author name, and date
  • Accent colour picker for lines, circles, and decorative elements
  • Live preview inside an iframe (no download needed)
  • One‑click download as a high‑quality A4 PDF
  • Responsive layout, works on desktop & tablet
  • Two logo support with college name centered (Final Cover Page)
  • Uploadable decorative line image (Final Cover Page)

🛠️ How to use

  1. Choose either Standard Cover Page or Final Cover Page from the buttons above.
  2. Fill in the fields on the left side of the generator page.
  3. Click Preview PDF to see the result instantly.
  4. If satisfied, click Download PDF to save the cover page.
  5. For the Final Cover Page, you can upload two logos (left and right) and a decorative project line image.

📄 Technical stack

HTML5, CSS3, vanilla JavaScript, and jsPDF v2.5.1. The PDF is generated in vector format and remains crisp at any zoom level.

🚀 Future plans

More cover templates, background patterns, additional font choices, and export options (PNG, SVG). Contributions are welcome!
